# **бҹЎбҹЎ. бһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸ**

бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ“бҹҒбҹҮбһӮбһәбһӣбҹҒбһҒбһ•бҹ’бһӣбһјбһңбһӣбҹҒбһ…бһҳбһ¶бһ“бһ“бһјбһңбһӣбҹҒбһ… бһ–бһёбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҠбһӣбҹӢбһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢбҹ”

**бҹ бҹЎ.** бһҡбҹҖбһ“бһҖбҹ’бһ“бһ»бһ„бһӣбҹҒбһҒбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ C# бһ“бһ·бһ„ .NET
бһҹбҹ’бһ‘бһ¶бһҖбҹӢбһҹбҹ’бһ‘бһҫбһҡбһӣбһҖбҹ’бһҒбһҺбҹҲбһҹбһҳбҹ’бһ”бһҸбҹ’бһҸбһ·бһӣбҹҒбһ… C#, OOP, async programming, бһ“бһ·бһ„бһӣбҹҒбһ…бһӣбһҳбҹүбһ¶бһҹбҹҠбһёбһ“ .NET бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҹбҹ’бһ‘бһ¶бһҖбҹӢбһҹбҹ’бһ‘бһҫбһҡбһӣбҹҒбһ…бҹ”

**бҹ бҹў.** бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһӣбһҳбҹ’бһўбһ·бһҸ ASP.NET Core

**бҹ бҹЈ.** бһ”бһ„бҹ’бһҖбһҫбһҸбһӣбҹҒбһҒ MVC бһ“бһ·бһ„ Web API Projects

**бҹ бҹӨ.** бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ MySQL
бһҹбҹ’бһ‘бһ¶бһҖбҹӢбһҹбҹ’бһ‘бһҫбһҡбһӣбҹҒбһ… EF Core, Dapper, бһ“бһ·бһ„ ADO.NET бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӣбҹҒбһ…бҹ’бһҹбһҶбҹ’бһӣбһ„бһӣбһҖбҹ’бһҒбһҺбҹҲбһӣбҹҒбһ…бҹ”

**бҹ бҹҘ.** бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҮбҹҶбһ“бһ¶бһүбһӣбҹҒбһ… Frontend Integration
